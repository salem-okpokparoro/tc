Built around what serious MT5 copying needs.

TCync stands out through the way its real app logic works: encrypted local sessions, licensed signal validation, broker symbol mapping, Live Only trade filtering, MasterCtrl close control and clear output logs.

Fast local copying

Runs locally with MetaTrader 5, detecting master trades and sending matching orders to configured slave accounts.

Slave Volume mode

Use each slave account’s saved lot size, or copy the master trade volume when you prefer direct mirroring.

Copy SL/TP

Choose whether Stop Loss and Take Profit are copied from the master trade to slave trades where supported.

Symbol exclusions

Exclude symbols by adding them to a list of excluded symbols: trades will not be copied for these symbols.

MasterCtrl

When enabled, slave trades close when the matching master trade closes. Turn it off when you want manual control.

Live Only

Start clean by skipping trades that were already open before Run was clicked, using the same style of skip logic as exclusions.

Signal channels

License-controlled channels can validate a channel password, expiry date and decrypted main credential before use.

Session logs

Save output logs to timestamped files under tcync_session/outputs, then clear the screen automatically.

Encryption enabled

Trading credentials are encrypted on local machine. End-to-end encryption enbaled for signal providers.

TCync is designed so saved trading credentials are protected through encrypted local session storage and only decrypted by the desktop app when connecting to MT5. Sensitive values are not printed into output logs, and signal-channel credentials are validated before use.

Privacy note: Keep your license key, Windows account, machine and local session folder secure. Encryption protects stored files, but users are still responsible for device security.

From install to copying in minutes.

Designed for a clear desktop workflow: activate, save accounts, set controls, enable MT5 algo trading, then run.

Activate license

Enter your license key, accept terms, and view your TCyncID directly from the app when support needs it.

Save main and slaves

Add the master account and licensed slave accounts. The app enforces the maximum slave count allowed by the license.

Enable MT5 algo trading

In MT5, go to Tools → Options → Expert Advisors and tick Allow algorithmic trading for each terminal.

Pick controls

Set Slave Volume, Copy SL/TP, MasterCtrl, Live Only, exclusions and optional Signal Channel.

Click Run

TCync initializes accounts, maps symbols between brokers and starts watching the master account for new trades.

Review logs

See account connection status, mapping, skipped exclusions, copied trades, close events and errors in the output window.
